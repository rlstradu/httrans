📖 Guía de uso e instalación

¿Qué versión elegir?

  • Gmail / Webmail: Utiliza iconos SVG más nítidos. Ideal para navegadores web y aplicaciones móviles modernas.
  • Outlook / Desktop: Utiliza caracteres estándar para iconos. Necesaria para Microsoft Outlook (escritorio), Apple Mail y entornos corporativos restrictivos.

Pasos de instalación

  1. Configura tus datos personales y diseño.
  2. Pulsa el botón "Copiar firma al portapapeles".
  3. Ve a la configuración de tu gestor de correo.
  4. Pega directamente en el cuadro de edición con Ctrl + V (o Cmd + V).
  5. Guarda los cambios.
⚠️ ¿Algo no se ve bien?

Si tras enviar un correo no aparecen los iconos, la foto o el marco, el cliente de correo del destinatario podría estar bloqueando el diseño avanzado.

Solución: Prueba a desmarcar las casillas de diseño (quitar marco o iconos) para obtener una versión más simplificada. Recomendamos siempre enviar un correo de prueba para verificar el resultado.
